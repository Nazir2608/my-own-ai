# MY-OWN-AI — Vector Database in Java

A production-grade Vector Database built from scratch in **Java 21 + Spring Boot** with:

- ✅ **3 Search Algorithms**: HNSW, KD-Tree, Brute Force (side-by-side comparison)
- ✅ **3 Distance Metrics**: Cosine, Euclidean, Manhattan
- ✅ **16D Demo Vectors**: Pre-loaded across 4 semantic categories
- ✅ **Real Document Embeddings**: Via Ollama's `nomic-embed-text` (768D)
- ✅ **RAG Pipeline**: Semantic search + LLM generation with local models
- ✅ **Interactive Web UI**: PCA visualization, live search, benchmarks
